# Two-Point Calculation Of \(f_{B_s^\ast}\)

For the vector current

\[
j_\mu=\bar s\gamma_\mu b
\]

we define

\[
\langle0|j_\mu|B_s^\ast(q,\epsilon)\rangle
=m_{B_s^\ast}f_{B_s^\ast}\epsilon_\mu .
\]

The LO transverse vector-current sum rule uses

\[
\rho_T^{\rm LO}(s)=
\frac{1}{8\pi^2}s(1-z)^2(2+z),
\qquad z=\frac{m_b^2}{s}.
\]

The script

```bash
python3 Bsstar_gamma/scripts/twopoint_vector_decay_constant_bsstar.py
```

scans

\[
5.0\le M^2\le8.0~{\rm GeV}^2,\qquad
s_0=33,34,35,36~{\rm GeV}^2.
\]

The scanned envelope is

\[
f_{B_s^\ast}=0.241\ldots0.283~{\rm GeV}.
\]

At the representative point

\[
M^2=6.5~{\rm GeV}^2,\qquad s_0=34.5~{\rm GeV}^2,
\]

we obtain

\[
f_{B_s^\ast}=0.2615~{\rm GeV}.
\]

For the radiative calculation we assign

\[
f_{B_s^\ast}=0.2615\pm0.0336~{\rm GeV}.
\]
