# Bs1 -> Bs* gamma

This folder contains the bottom-strange vector-final-state continuation of the
`Dsstar_gamma` calculation.  The same tensor basis and calibrated hard/soft
LCSR machinery are used with

\[
c\to b,\qquad D_s^\ast\to B_s^\ast,\qquad D_{s1}\to B_{s1}.
\]

The numerical outputs are written under `Bsstar_gamma/outputs`, and the
calculation notes are written under `Bsstar_gamma/notes`.
