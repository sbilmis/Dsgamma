# Notes Folder

This folder is now the active working area for the calculation notes, redo
analysis, publication-quality plot planning, and internal reports.

- `Ds1_to_Ds_gamma_LCSR_notes.tex` - active derivation, calculation, and
  redo-analysis notes.
- `Ds1_to_Ds_gamma_LCSR_notes.pdf` - compiled full notes.
- `pre_automode_report.tex` - early roadmap/report snapshot.
- `DsBs_gamma_Mathematica_derivation.tex` - clean, standalone derivation of
  the independent Mathematica/FeynCalc restart.
- `DsBs_gamma_Mathematica_derivation.pdf` - compiled standalone derivation.
- `internal_future_work.md` - private follow-up ideas not intended for the
  present paper.

The notes are intentionally more detailed than the paper draft and include
intermediate checks, diagnostic tables, derivations useful for students, and the
new section on redo analysis and publication-quality plots.

The Mathematica restart is intentionally isolated in
`DsBs_gamma_Mathematica_derivation.*`; it should not be folded into the long
numerical working notes while the symbolic derivation is still in progress.
