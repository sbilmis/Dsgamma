Mathematica/FeynCalc current, Wick-contraction, and trace audit
================================================================

Scope: currents -> Wick contraction -> propagators -> traces -> E1 projection.
Loop integration, spectral densities, and Borel transforms are not performed here.

1. CURRENT DEFINITIONS
J_A,mu = sbar gamma_mu gamma5 Q
J_B,mu = i sbar sigma_(mu alpha) pPrime^alpha gamma5 Q/(mQ+ms)
Correlator: T{J_P^dagger(x) J_X,mu(0)} with J_P^dagger=sbar i gamma5 Q.
pPrime = p+q; q^2=0 is imposed only in the E1 projection.
Physical-current rotations are linear combinations of J_A and J_B.

2. WICK CONTRACTION AND GLOBAL SIGN
Hard double contraction:
  <T{sbar GammaP Q Qbar Gamma_X s}> = -Tr[GammaP S_Q Gamma_X S_s].
Hard correlator prefactor before propagator/vertex factors (-i Nc):
(-I)*Nc

Soft correlator prefactor before Fierz (+i):
I

Soft correlator prefactor after the -1/4 Fierz factor:
-1/4*I

The old numerator files intentionally omitted these overall factors.

3. PROPAGATOR ROUTING
Free propagator: S_f^0(r)=i (slash(r)+m_f)/(r^2-m_f^2+i0).
One-photon insertion is oriented separately on each fermion line; see the routing list below.
dH1 = (k+q)^2 - mQ^2
dH2 = k^2 - mQ^2
dH3 = (k-p)^2 - ms^2
dS1 = k^2 - mQ^2
dS2 = (k-p)^2 - ms^2
dS3 = (k-p-q)^2 - ms^2
